# ❤️ The Josh & KJ Decision Machine

Fresh mobile-first PWA roulette for dates and food.

## GitHub Pages setup
Upload these files to the root of your repo:
- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

## Music behavior
This version uses built-in generated background music, so no MP3 is required.
It tries to play when the app opens. Some phone browsers block sound until the first tap, so the app shows a full-screen “Start the Machine” button when needed. After that, the 🔊 / 🔇 button controls music and remembers your choice.
